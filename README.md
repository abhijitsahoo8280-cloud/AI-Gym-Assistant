# AI Gym & Fitness Assistant 

A real-time AI application built with Python to track gym workouts and provide diet plans.

## Features
- *Bicep Curl Counter:* Real-time repetition counting using MediaPipe.
- *AI Dietician:* BMI calculation and personalized diet suggestions.
- *Interactive UI:* Built with Streamlit.

## Tech Stack
- Python, OpenCV, MediaPipe, Streamlit
